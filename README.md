## 📖 О порте

HBM's Nuclear Tech Mod — один из самых масштабных технических модов для Minecraft 1.7.10, добавляющий сотни механизмов, ядерные реакторы, баллистические ракеты, радиацию, загрязнение, обширную систему вооружения и многое другое. Цель этого репозитория — перенести всё это богатство на актуальную версию игры с сохранением оригинального геймплея и расширением возможностей за счёт новых механик Forge.

**Ключевые особенности портирования:**
- Полная переработка кодовой базы под архитектуру 1.20.1 (регистры через `DeferredRegister`, использование `IEventBus`, переписанный рендеринг и сетевое взаимодействие).
- Адаптация всех конфигов в TOML-формат с разделением на клиент/сервер/общие.
- Сохранение уникальных систем: радиация, загрязнение, нейтронная физика, многоуровневая система опасностей (`HazardSystem`).
- Портирование продвинутого контента: ракеты (включая Tier 0–4), кастомные ядерные взрывы, сущности вроде `BlackHole`, `RagingVortex`, `Glyphid` (вдохновлённые Deep Rock Galactic), `FBI`, `BOT Prime` и многие другие.

## 🔧 Сборка из исходников (для разработчиков)

1. Установите JDK 17 и Git.
2. Клонируйте репозиторий (когда он станет публичным):
   ```bash
   git clone https://github.com/ваш-логин/hbm-ntm-1.20.1.git
3. Откройте папку проекта в IDE, поддерживающей Gradle (IntelliJ IDEA, Eclipse).
4. Выполните задачу gradlew build. Скомпилированный jar появится в build/libs.
5. Для запуска в среде разработки используйте стандартные запуски Forge (runClient, runServer).
