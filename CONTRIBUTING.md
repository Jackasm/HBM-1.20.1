# Contributing to HBM 1.20.1 Port

Спасибо за интерес к проекту! На данный момент порт находится в стадии закрытого тестирования.

## Как помочь сейчас

- **Тестирование:** Публичное тестирование откроется позже. Следите за анонсами в разделе Releases.
- **Баг-репорты:** Пожалуйста, создавайте Issue с шаблоном "Bug Report".
- **Предложения:** Используйте шаблон "Feature Request" для идей по портированию.

## Правила

- Будьте вежливы и конструктивны.
- Перед созданием нового Issue проверьте, нет ли уже похожего.
- Pull requests пока не принимаются (репозиторий в процессе настройки CI/CD).

## Связь

Если вы хотите обсудить сотрудничество, создайте Issue с меткой `[Discussion]`.
